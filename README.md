# CRM Systém - Kmeňová produkcia

Webová aplikácia pre správu klientov a finančných produktov s možnosťou importu CSV dát.

## Funkcie
- **Autentifikácia**: Registrácia a prihlásenie používateľov
- **Import CSV**: Načítanie a spracovanie CSV súborov s dátami o zmluvách
- **Dashboard**: Prehľad štatistík a grafov
- **Správa klientov**: Zoznam klientov s detailnými informáciami
- **Príležitosti**: Cross-selling a up-selling doporučenia
- **Kalendár**: Plánovanie schôdzok s klientmi
- **Kategórie produktov**: 
  - Úvery (UV)
  - Investície (F, FS)
  - Životné poistenie (ZH, ZP)
  - Majetkové poistenie (PZP, DBD, KAS, N)

## Technológie
- HTML5, CSS3, JavaScript (ES6+)
- Chart.js pre grafy
- Font Awesome ikony
- Local Storage pre ukladanie dát
